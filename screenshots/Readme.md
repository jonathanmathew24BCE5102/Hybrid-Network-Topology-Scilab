# Hybrid Network Topology using Scilab

This repository contains a hybrid network topology created using:
- Star topology
- Ring topology
- Mesh topology

## Software Used
- Scilab 6.0.0
- NARVAL Toolbox

## Files
- hybrid_topology.sce → Scilab source code
- screenshots/ → Output screenshots

## Result
A hybrid topology with 25 nodes was successfully created and analyzed.
